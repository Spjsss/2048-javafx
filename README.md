2048 JavaFX Game

A desktop implementation of the classic 2048 puzzle game built using JavaFX.
The game allows players to combine tiles with the same number to achieve higher scores.

🎮 Features
Grid-based game interface
Tile movement in four directions (up, down, left, right)
Tile merging logic
Score tracking system
Smooth user interaction with keyboard input
🛠️ Technologies Used
Java
JavaFX
🚀 How to Run
1. Clone the repository
git clone https://github.com/Spjsss/2048-javafx.git
2. Open the project

Open the project in an IDE such as IntelliJ IDEA, NetBeans, or VS Code

3. Configure JavaFX

Make sure JavaFX is installed and added to your project.

Example VM options:

--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
4. Run the application

Run the main class to start the game
